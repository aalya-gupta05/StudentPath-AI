# StudentPath-AI 🎓

> Built for the **Microsoft Agents League Hackathon 2026** during Microsoft AI Skill Fest 2026

An intelligent AI Career Guidance Agent built on **Microsoft Azure AI Foundry** that helps students navigate their career journey — from self-discovery to job readiness.

---

## 🤖 What it does

StudentPath-AI is a conversational agent that guides students through:

- **Personality & interest assessment** — understands who you are before suggesting what to do
- **Career path exploration** — maps out realistic options based on your background
- **Skill gap analysis** — identifies what you need to learn to reach your goal
- **Roadmap creation** — builds a step-by-step learning and growth plan
- **Resume & interview prep** — practical advice for getting hired
- **Mental health & motivation support** — keeps students encouraged through the process

## 💡 How it was built

StudentPath-AI was created using **Azure AI Foundry's agent builder** — no traditional code required. The agent was configured with:
- A custom system prompt defining its persona and scope
- Multi-turn conversation flow for guided career counselling
- Azure's hosted GPT-oss-120b model for natural language understanding

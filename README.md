# StudentPath-AI

An intelligent AI Career Guidance Agent built using Microsoft Azure AI Foundry.

- Personalized career guidance for students
- Psychological personality assessment
- Skill gap analysis & roadmap
- Resume & interview preparation
- Mental health & motivation support
